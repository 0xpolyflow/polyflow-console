Gas Station
Watch Demo
The gas station component activates when Polyflow detects a user is trying to execute a transaction for which they don't have enough gas.
It will then offer multiple ways for the user to fill their wallets with gas.
Network Switcher
Watch Demo
The network switcher component detects when the network the user is connected to in their wallet provider doensn't match the network of the transaction. It will then prompt the user to switch the network.

Wallet connect component provides an easy-to-implement solution to connect your users to many popular wallet solutions.
Compliance
Watch Demo
Compliance component detects when users from forbidden countries or address blacklists try to execute transactions on your frontend. It can stop them from executing transactions or require signing a compliance disclaimer.
Error Messages
Watch Demo
Error messages module detects when a transaction will result in a revert. It will then parse the error message and display a popup, guiding the user how to sucessfully execute a transaction.
